# 💪 PPL Weight Tracker

A 6-day Push Pull Legs workout weight tracker built with HTML/CSS/JS.

## 🔗 Live Tracker
👉 **[OPEN TRACKER HERE](https://jatothubichu12-cpu.github.io/ppl-tracker/PPL_Weight_Tracker.html)**

## 📋 Program
| Day | Session | Focus |
|-----|---------|-------|
| Monday | Push 1 | Chest / Shoulders / Triceps |
| Tuesday | Pull 1 | Back / Biceps / Rear Delts |
| Wednesday | Legs 1 | Quads / Hamstrings / Glutes |
| Thursday | Push 2 | Chest / Shoulders / Triceps |
| Friday | Pull 2 | Back / Biceps / Rear Delts |
| Saturday | Legs 2 | Quads / Hamstrings / Glutes |
| Sunday | Rest | Full Recovery |

## ✅ Features
- Track weights for every set of every exercise
- Week by week progress tracking
- See progress vs previous week
- Weight history for all weeks
- Export data as CSV
- Auto-saves in browser

## 📱 Mobile
Add to home screen on Android/iPhone for quick gym access.
